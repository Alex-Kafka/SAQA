# WebSocket

**WebSocket** — это протокол связи, обеспечивающий постоянное, двустороннее (duplex) соединение между клиентом и сервером поверх одного TCP-соединения.

### Как работает?
1.  **Handshake:** Клиент отправляет HTTP-запрос с заголовком `Upgrade: websocket`.
2.  **Установка соединения:** Сервер подтверждает обновление протокола.
3.  **Обмен данными:** Клиент и сервер могут обмениваться сообщениями в реальном времени без необходимости постоянно устанавливать новые соединения.

### Отличие от HTTP
- **HTTP:** Запрос-ответ, stateless, клиент всегда инициирует.
- **WebSocket:** Постоянное соединение, двусторонняя связь, сервер может отправлять данные в любой момент.

### Когда использовать?
- Чат-приложения
- Онлайн-игры
- Торговые платформы
- Совместная работа в реальном времени
- Push-уведомления
