# Оптимизация БД

**Оптимизация базы данных** - это процесс улучшения производительности БД через различные методы.

### Способы оптимизации
1. **Индексы** - ускорение поиска данных.
2. **Нормализация** - устранение избыточности данных.
3. **Денормализация** - сознательное добавление избыточности для ускорения запросов.
4. **Партиционирование** - разделение таблицы на части по какому-либо признаку.
5. **Кэширование** - сохранение часто запрашиваемых данных в памяти.
6. **Оптимизация запросов** - переписывание запросов для более эффективного выполнения.

### Масштабирование БД
- **Вертикальное** - увеличение мощности сервера (CPU, RAM, Disk).
- **Горизонтальное** - распределение данных across several servers (шардирование).
