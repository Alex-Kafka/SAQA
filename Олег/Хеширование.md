# Хеширование

**Хеширование** - это преобразование входных данных произвольной длины в битовую строку фиксированной длины (хеш) с помощью хеш-функции.

### Свойства хорошей хеш-функции
- **Детерминированность:** Один и тот же вход всегда дает одинаковый хеш.
- **Быстрота:** Вычисление хеша происходит быстро.
- **Устойчивость к коллизиям:** Трудно найти два разных входа с одинаковым хешем.
- **Необратимость:** По хешу нельзя восстановить исходные данные.

### Примеры алгоритмов
- **MD5** (устарел, ненадежен)
- **SHA-1** (устарел, ненадежен)
- **SHA-256** (надежный, используется в Bitcoin)
- **bcrypt** (для паролей)

### Применение
- Хранение паролей (в хешированном виде)
- Проверка целостности данных
- Цифровые подписи
