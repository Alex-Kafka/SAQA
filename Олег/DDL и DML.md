

# DDL и DML

**DDL (Data Definition Language)** - язык определения данных. Команды для создания, изменения и удаления структур данных (таблиц, схем, индексов).

**DML (Data Manipulation Language)** - язык манипуляции данными. Команды для работы с данными (вставка, выборка, обновление, удаление).

### Команды DDL
- **CREATE** - создает объекты (таблицу, индекс, базу данных)
- **ALTER** - изменяет структуру объектов
- **DROP** - удаляет объекты
- **TRUNCATE** - удаляет все данные из таблицы, но не саму таблицу

### Команды DML
- **SELECT** - выбирает данные
- **INSERT** - вставляет новые данные
- **UPDATE** - обновляет существующие данные
- **DELETE** - удаляет данные
