# Симметричное и асимметричное шифрование

**Симметричное шифрование** - один ключ для шифрования и дешифрования.

**Асимметричное шифрование** - два ключа: открытый (public) для шифрования и закрытый (private) для дешифрования.

### Сравнение
| Параметр | Симметричное | Асимметричное |
|----------|--------------|---------------|
| **Количество ключей** | Один | Два (публичный и приватный) |
| **Скорость** | Быстро | Медленно |
| **Безопасность** | Зависит от сохранности ключа | Зависит от сохранности приватного ключа |
| **Использование** | Шифрование больших объемов данных | Обмен симметричными ключами, цифровые подписи |

### Примеры
- **Симметричное:** AES, DES
- **Асимметричное:** RSA, ECC

### Применение в HTTPS
Асимметричное шифрование используется для установки сеанса и обмена симметричным ключом, а затем симметричное шифрование для обмена данными.
