# SQL Основы

**SQL (Structured Query Language)** — это язык структурированных запросов, используемый для работы с реляционными базами данных.

### Основные команды SQL
- **SELECT** - выборка данных
- **INSERT** - вставка новых данных
- **UPDATE** - обновление существующих данных
- **DELETE** - удаление данных
- **CREATE** - создание объектов (таблиц, индексов и т.д.)
- **ALTER** - изменение структуры объектов
- **DROP** - удаление объектов

### Оператор SELECT
Базовая структура запроса SELECT:
```sql
SELECT [столбцы]
FROM [таблица]
[WHERE условия]
[GROUP BY столбцы]
[HAVING условия]
[ORDER BY столбцы]
