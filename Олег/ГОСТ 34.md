# ГОСТ 34

**ГОСТ 34.XXX — Информационные технологии (ИТ) — Комплекс стандартов на автоматизированные системы (АС)** — это набор стандартов, регулирующих разработку автоматизированных систем (более широкое понятие, чем просто ПО).

### Для чего нужен?
- Регламентирует процессы создания автоматизированных систем, включая ПО,硬件, организационные вопросы.
- Применяется при разработке сложных информационных систем, особенно в государственном секторе.

### Основные документы по ГОСТ 34
- **Техническое задание (ТЗ)** по ГОСТ 34.602-89 — аналог [[ТЗ (Техническое задание)|ТЗ]].
- **Технический проект** — по ГОСТ 34.201-89.
- **Рабочая документация** — по ГОСТ 34.201-89.

### Разница между ГОСТ 19 и ГОСТ 34
- **ГОСТ 19** фокусируется на документации для **программного обеспечения**.
- **ГОСТ 34** охватывает всю **автоматизированную систему** (включая аппаратное обеспечение, организационные структуры, обучение и т.д.).
