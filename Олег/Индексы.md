# Индексы

**Индекс** - это структура данных, которая ускоряет операции извлечения данных из таблицы за счет дополнительных затрат на запись и хранение.

### Типы индексов
- **B-дерево** - наиболее распространенный, подходит для диапазонных запросов.
- **Хеш-индекс** - для точных совпадений.
- **Bitmap** - для столбцов с небольшим количеством уникальных значений.
- **GIN** - для составных данных (массивы, JSON, полнотекстовый поиск).

### Когда использовать?
- Столбцы, часто используемые в WHERE.
- Столбцы, используемые в JOIN.
- Столбцы, используемые в ORDER BY.

### Когда не использовать?
- В таблицах с частыми операциями вставки/обновления (замедляет запись).
- В таблицах с небольшим объемом данных.
