# FTP и SFTP

**FTP (File Transfer Protocol)** - протокол передачи файлов по сети. Использует два канала: управляющий (порт 21) и данных (порт 20).

**SFTP (SSH File Transfer Protocol)** - протокол передачи файсов, работающий поверх SSH (порт 22). Обеспечивает безопасную передачу.

### Различия
| Параметр | FTP | SFTP |
|----------|-----|------|
| **Безопасность** | Данные передаются в открытом виде | Шифрование и аутентификация через SSH |
| **Порты** | 21 (управление) и 20 (данные) | 22 |
| **Протокол** | Отдельный протокол | Работает поверх SSH |
| **Передача данных** | Может быть проблемы с брандмауэром | Легко проходит через брандмауэр |

### Когда использовать?
- **FTP** - для неконфиденциальных данных в внутренней сети.
- **SFTP** - для передачи конфиденциальных данных.
