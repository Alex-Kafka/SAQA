# HTTPS

**HTTPS (HyperText Transfer Protocol Secure)** - защищенная версия HTTP, использующая шифрование TLS/SSL.

### Как работает?
1. **Клиент** запрашивает защищенное соединение.
2. **Сервер** отправляет свой SSL-сертификат.
3. **Клиент** проверяет сертификат.
4. **Клиент и сервер** устанавливают симметричный сеансовый ключ через асимметричное шифрование.
5. **Данные** передаются через симметричное шифрование.

### Зачем нужен?
- Шифрование данных между клиентом и сервером.
- Аутентификация сервера (клиент уверен, что общается с нужным сервером).
- Целостность данных (данные не были изменены при передаче).
