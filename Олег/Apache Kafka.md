# Apache Kafka

**Apache Kafka** - распределенная платформа потоковой обработки данных, работающая по принципу публикация-подписка.

### Основные понятия
- **Топик (Topic)** - категория или поток сообщений.
- **Партиция (Partition)** - топик делится на партиции для параллельной обработки.
- **Producer** - приложение, отправляющее сообщения в топик.
- **Consumer** - приложение, читающее сообщения из топика.
- **Consumer Group** - группа потребителей, совместно обрабатывающих сообщения.

### Особенности
- Сообщения сохраняются на диске и реплицируются.
- Поддержка перечитывания сообщений (consumer может сбросить offset).
- Высокая пропускная способность и отказоустойчивость.
