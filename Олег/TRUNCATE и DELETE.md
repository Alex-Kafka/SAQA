# TRUNCATE и DELETE

**DELETE** - оператор DML, который удаляет строки из таблицы по условию WHERE. Если условие не указано, удаляет все строки.

**TRUNCATE** - оператор DDL, который удаляет все строки из таблицы, освобождая место в базе данных.

### Различия
| Параметр | DELETE | TRUNCATE |
|----------|--------|----------|
| **Скорость** | Медленнее, так как записывает в лог каждую удаленную строку | Быстрее, так как удаляет данные без записи в лог отдельных строк |
| **WHERE** | Может использоваться с условием | Не может использоваться с условием (всегда удаляет все) |
| **Триггеры** | Запускает триггеры удаления | Не запускает триггеры |
| **Откат (ROLLBACK)** | Можно откатить в транзакции | В большинстве СУБД нельзя откатить |
| **Автоинкремент** | Не сбрасывает счетчик автоинкремента | Сбрасывает счетчик автоинкремента |

### Примеры

-- Удалить все строки
DELETE FROM table_name;

-- Удалить по условию
DELETE FROM table_name WHERE condition;

-- Очистить таблицу полностью
TRUNCATE TABLE table_name;
